# SSL
    * SSL证书就是指遵守SSL安全套阶层协议的服务器数字证书（SercureSocketLayer）
    * 美国网景公司开发
    * CA（CertifacateAuthority）是数字证书认证中心，是发放，管理，废除数字证书的授信人的第三方机构
    
* 如果SSL证书是未授权状态，浏览器都会出现警告，但是可以使用代码可以忽略掉

* js 加密
    * 有的反爬虫策略采用的是JS对需要传输的数据进行加密处理(通常是取md5加密算法)
